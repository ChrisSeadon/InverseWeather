# Inverse Weather (WIP)
A simple application that compares weather in the user's location with the weather at their antipode.

## Tech stack
- Front-end: Next-js/React
- Back-end: Node.js/Next.js App Router
- Map library: Leaflet JS
- Weather API: OpenWeatherMap
- CSS Framework: Tailwind CSS
- Hosting Platform: Vercel

The application is deployed at: [inverse-weather.vercel.app](inverse-weather.vercel.app)
